Este projeto consiste em uma aplicação web desenvolvida utilizando Vue 3 com TypeScript, cujo objetivo é facilitar o gerenciamento e acompanhamento de tarefas entre desenvolvedores dentro de uma equipe.

A aplicação permite organizar atividades, visualizar o progresso das tarefas e distribuir demandas entre membros do time de desenvolvimento, promovendo maior controle, transparência e eficiência no fluxo de trabalho.

O sistema foi projetado com foco em simplicidade, organização e boa experiência de uso, permitindo que os desenvolvedores acompanhem rapidamente o status de suas tarefas e as responsabilidades de cada membro da equipe.
